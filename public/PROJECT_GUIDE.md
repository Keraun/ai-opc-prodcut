## 欢迎使用

欢迎使用本网站管理系统！本手册将帮助您快速上手，轻松管理您的网站内容。

### 系统概览
本系统提供直观的可视化界面，让您无需编程知识即可：
- 创建和编辑网站页面
- 发布产品和文章
- 管理图片资源
- 自定义网站主题
- 查看用户留言

## 登录后台

### 如何登录
1. 在浏览器地址栏输入网站地址，后面加上 `/admin`（例如：`https://您的网站.com/admin`）
2. 输入您的用户名和密码
3. 点击「登录」按钮

### 首次登录
- 首次登录后，系统会提示您修改初始密码
- 请设置一个安全的密码，并妥善保管
- 建议定期更换密码以保障账号安全

### 退出登录
点击页面右上角的头像或用户名，选择「退出登录」即可安全退出。

## 全局配置

「全局配置」是设置网站基本信息的地方，在这里您可以：

### 网站基本信息
- **网站名称**：显示在浏览器标签页和网站标题处
- **网站描述**：简短介绍网站的用途
- **网站 Logo**：上传您的品牌 Logo
- **联系信息**：设置电话、邮箱、地址等联系方式

### SEO 优化
- **关键词**：设置与网站相关的关键词，帮助搜索引擎收录
- **页面描述**：设置网站在搜索结果中显示的描述文字
- **社交媒体配置**：设置 Open Graph 和 Twitter Card 信息，优化分享效果

### 统计代码
- **百度统计**：输入百度统计代码，追踪网站访问数据
- **谷歌分析**：输入谷歌分析 ID，获取更详细的数据分析

## 页面管理

「页面管理」是构建网站的核心功能，您可以创建、编辑和管理所有页面。

### 创建新页面
1. 点击左侧菜单的「页面管理」
2. 点击「新建页面」按钮
3. 输入页面名称和路径
4. 点击「确定」创建页面

### 编辑页面
1. 在页面列表中找到要编辑的页面
2. 点击「编辑」按钮进入编辑器
3. 在编辑器中可以：
   - **添加模块**：点击「添加模块」选择想要的模块类型
   - **编辑模块**：点击模块上的编辑图标修改内容
   - **拖拽排序**：按住模块拖动来调整顺序
   - **删除模块**：点击模块上的删除图标移除模块

### 常用模块介绍
- **导航栏**：网站顶部的导航菜单
- **Hero 横幅**：首页顶部的大幅宣传区域
- **产品展示**：展示产品列表的模块
- **文章列表**：显示最新资讯的模块
- **关于我们**：介绍公司或团队的模块
- **联系方式**：展示联系表单和信息的模块
- **页脚**：网站底部的版权和链接信息

### 预览和发布
- **实时预览**：编辑时点击「预览」按钮查看效果
- **保存**：点击「保存」保存当前修改（不会立即发布）
- **发布**：点击「发布」将更改上线到网站
- **下架**：点击「下架」将页面从网站隐藏

## 产品管理

「产品管理」用于管理您的产品信息。

### 添加产品分类
1. 点击「产品管理」
2. 切换到「分类管理」标签
3. 点击「添加分类」
4. 输入分类名称和排序
5. 点击「保存」

### 添加新产品
1. 在「产品管理」页面点击「添加产品」
2. 填写产品信息：
   - **产品名称**：产品的标题
   - **产品分类**：选择所属分类
   - **产品图片**：上传产品主图和多张详情图
   - **产品描述**：详细介绍产品功能和特点
   - **价格**：设置产品价格
   - **排序**：调整产品显示顺序
3. 点击「保存」

### 产品上下架
- **上架**：在产品列表中点击「上架」，产品将在网站显示
- **下架**：点击「下架」，产品将从网站隐藏

### 编辑和删除
- 点击「编辑」修改产品信息
- 点击「删除」移除产品（删除后不可恢复，请谨慎操作）

## 资讯管理

「资讯管理」用于发布新闻、公告、博客文章等内容。

### 发布新文章
1. 点击「资讯管理」
2. 点击「发布文章」
3. 填写文章信息：
   - **文章标题**：文章的标题
   - **封面图片**：上传文章封面图
   - **文章摘要**：简短的文章介绍
   - **文章内容**：使用富文本编辑器编写详细内容
   - **发布时间**：设置发布时间
4. 点击「发布」

### 富文本编辑器使用
编辑器支持以下功能：
- **文字格式**：加粗、斜体、下划线、删除线
- **标题**：H1-H6 标题样式
- **列表**：有序列表和无序列表
- **链接**：添加网页链接
- **图片**：插入图片
- **引用**：引用样式
- **对齐**：左对齐、居中、右对齐

### 管理文章
- 在文章列表中可以查看所有文章
- 点击「编辑」修改文章内容
- 点击「删除」移除文章
- 可以按发布时间筛选和排序

## 留言管理

「留言管理」用于查看和管理用户通过网站提交的留言。

### 查看留言
1. 点击「留言管理」
2. 可以看到所有用户留言列表
3. 每条留言显示：
   - 用户姓名
   - 联系邮箱
   - 留言内容
   - 提交时间
4. 点击留言可以查看详情

### 消息推送
系统支持向用户推送消息通知：
1. 点击「推送消息」
2. 填写消息标题和内容
3. 选择推送对象
4. 点击「发送」

### 推送记录
在「推送记录」标签中可以查看所有已发送的消息，包括：
- 发送时间
- 消息内容
- 发送状态
- 可以重试失败的推送

## 图片管理

「图片管理」用于统一管理网站使用的所有图片资源。

### 上传图片
1. 点击「图片管理」
2. 点击「上传图片」按钮
3. 选择要上传的图片文件（支持 JPG、PNG、GIF、WebP 格式）
4. 可以同时上传多张图片
5. 等待上传完成

### 图片分类
- 可以创建图片分类，方便管理
- 上传时选择分类，或后期移动图片到不同分类

### 使用图片
- 在页面编辑、产品编辑、文章编辑时，可以从图片库选择已上传的图片
- 也可以直接上传新图片

### 删除图片
- 勾选要删除的图片
- 点击「删除」按钮
- 注意：删除的图片无法恢复，请谨慎操作

## 主题管理

「主题管理」用于自定义网站的外观风格。

### 切换主题
1. 点击「主题管理」
2. 可以看到预设的主题列表
3. 点击「应用」按钮切换到想要的主题
4. 刷新网站前端即可看到效果

### 自定义主题
如果预设主题不够用，您可以自定义主题：
1. 点击「自定义主题」
2. 设置颜色方案：
   - **主色**：网站主色调
   - **辅色**：辅助色
   - **文字色**：正文文字颜色
   - **背景色**：页面背景颜色
   - **成功/警告/错误色**：状态提示颜色
3. 设置效果：
   - **圆角**：按钮和卡片的圆角大小
   - **阴影**：阴影效果强度
4. 点击「保存」应用自定义主题

### 导入导出主题
- 可以导出当前主题配置，分享给他人或备份
- 可以导入主题配置文件，快速应用别人分享的主题

## 账号管理

「账号管理」用于管理后台登录账号。

### 修改密码
1. 点击右上角头像，选择「修改密码」
2. 输入当前密码
3. 输入新密码（建议至少 8 位，包含字母和数字）
4. 确认新密码
5. 点击「确定」

### 添加新账号
1. 点击「账号管理」
2. 点击「添加账号」
3. 填写账号信息：
   - **用户名**：登录用的用户名
   - **显示名称**：在后台显示的名称
   - **密码**：设置初始密码
   - **角色**：选择账号权限（管理员/普通用户）
4. 点击「保存」

### 管理账号
- **编辑**：修改账号信息和权限
- **重置密码**：为账号设置新密码
- **禁用/启用**：暂停或恢复账号登录权限
- **删除**：移除账号（删除后无法恢复）

### 登录日志
在「登录日志」标签中可以查看所有账号的登录记录，包括：
- 登录时间
- 登录 IP
- 浏览器信息
- 登录状态

## 系统管理

「系统管理」提供系统级的管理功能。

### 系统信息
- 查看系统版本
- 查看服务器状态
- 查看数据库大小
- 查看最后备份时间

### 系统操作
- **重启系统**：重启应用服务（慎用，会短暂中断服务）
- **清空缓存**：清除系统缓存，解决一些显示问题
- **重置配置**：将所有配置恢复到默认值（会丢失自定义配置，请先备份）

### 数据备份
- 建议定期备份数据
- 点击「备份数据」下载备份文件
- 保存好备份文件，以便需要时恢复

### 邮件配置
如果需要使用邮件功能（如密码重置、消息通知），需要配置 SMTP：
- **SMTP 服务器**：邮件服务器地址
- **端口**：SMTP 端口（通常是 465 或 587）
- **用户名**：邮箱账号
- **密码**：邮箱密码或授权码
- **发件人名称**：显示在邮件中的发件人名称
- 配置完成后可以点击「测试邮件」验证配置是否正确

## 常见问题

### 页面修改后网站没更新？
- 请确认是否点击了「发布」按钮，只「保存」不会上线
- 尝试刷新浏览器页面（Ctrl+F5 强制刷新）
- 在「系统管理」中点击「清空缓存」

### 图片上传失败？
- 检查图片大小是否超过限制（通常是 5MB）
- 确认图片格式是否支持（JPG、PNG、GIF、WebP）
- 尝试使用较小的图片重新上传

### 忘记密码怎么办？
- 联系超级管理员重置密码
- 如果配置了邮件功能，可以在登录页点击「忘记密码」通过邮件重置

### 如何让网站被搜索引擎收录？
- 在「全局配置」中完善 SEO 信息
- 保持网站内容定期更新
- 可以向搜索引擎提交网站地址
- 确保网站可以正常访问

### 数据会丢失吗？
- 系统会自动保存数据
- 建议定期在「系统管理」中备份数据
- 删除操作不可恢复，请谨慎操作

### 可以在手机上管理网站吗？
- 可以！后台管理界面支持响应式设计
- 在手机浏览器访问后台地址即可使用
- 建议在平板或电脑上进行复杂的页面编辑操作
